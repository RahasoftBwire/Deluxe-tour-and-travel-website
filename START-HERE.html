Project Setup Complete!
All files have been created successfully. Follow the steps below to run the application.

Step 1: Install Node.js

  1. Visit https://nodejs.org/
  2. Download the LTS (Long Term Support) version
  3. Run the installer with default settings
  4. Restart your computer (or at least PowerShell)
  5. Verify installation by opening PowerShell and typing:
node --version
npm --version

Step 2: Install Dependencies

Open PowerShell and navigate to the backend folder:

cd "c:\Users\Afronic\Desktop\Deluxe tour and travel website\backend"
npm install

This will install all required packages (Express, MongoDB, JWT, etc.)

Step 3: Install Frontend Server

npm install -g http-server

This installs a simple web server to serve the frontend files.

Step 4: Start the Application

Option A: Automated (Recommended)

  1. Navigate to the project folder
  2. Right-click on setup-and-test.ps1
  3. Select "Run with PowerShell"

Option B: Manual Start

Open TWO PowerShell windows:

Window 1 (Backend):

cd "c:\Users\Afronic\Desktop\Deluxe tour and travel website\backend"
npm run dev

Window 2 (Frontend):

cd "c:\Users\Afronic\Desktop\Deluxe tour and travel website\frontend"
npx http-server -p 3000 -o

Step 5: Access the Website

Once both servers are running:

  • Frontend: http://localhost:3000
  • Backend API: http://localhost:5000
