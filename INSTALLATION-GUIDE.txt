## STEP 1: Check Node.js Installation

Open PowerShell and run:
```
node --version
npm --version
```

If not installed:
1. Visit: https://nodejs.org/
2. Download LTS version (recommended)
3. Install with default settings
4. Restart PowerShell

## STEP 2: Automated Setup (RECOMMENDED)

Right-click on `setup-and-test.ps1` and select "Run with PowerShell"

OR run in PowerShell:
```powershell
cd "c:\Users\Afronic\Desktop\Deluxe tour and travel website"
.\setup-and-test.ps1
```

This script will:
- Check Node.js installation
- Install all backend dependencies
- Install http-server for frontend
- Offer to start both servers automatically

## STEP 3: Manual Setup (Alternative)

### Install Backend Dependencies:
```powershell
cd "c:\Users\Afronic\Desktop\Deluxe tour and travel website\backend"
npm install
```

### Install Frontend Server:
```powershell
npm install -g http-server
```

## STEP 4: Start the Application

### Option A: Using Quick Start Script

**Terminal 2 - Frontend:**
```powershell

```
Frontend runs on: http://localhost:3000

## STEP 5: Access the Website

1. Open your browser
2. Go to: http://localhost:3000
3. The website should load automatically!

## Troubleshooting

### If PowerShell script won't run:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### If port 3000 or 5000 is already in use:
Change the PORT in:
- Backend: `.env` file
- Frontend: use `http-server -p 3001`

### If MongoDB connection fails:
- Install MongoDB locally, OR
- Use MongoDB Atlas (cloud)
- Update MONGODB_URI in `.env`
