# Deluxe Tour & Travel - Installation Guide
# Quick setup for cloning on another machine

## STEP 1: Install Node.js
# Download and install from: https://nodejs.org/ (v22.20.0 or higher)

## STEP 2: Install MongoDB (Choose one option)
# Option A - Local: https://www.mongodb.com/try/download/community
# Option B - Cloud: https://www.mongodb.com/cloud/atlas (Free tier available)

## STEP 3: Install ALL dependencies with ONE command
# Open PowerShell/Terminal in project root and run:

cd backend && npm install && cd ../frontend && npm install && cd ..

## STEP 4: Configure Environment
# Create backend/.env file with these settings:

PORT=5000
NODE_ENV=development
MONGODB_URI=mongodb://localhost:27017/deluxe_tour_travel
JWT_SECRET=your_secret_key_change_in_production
JWT_EXPIRE=7d

## STEP 5: Run the application

# Terminal 1 - Backend:
cd backend
npm run dev

# Terminal 2 - Frontend:
cd frontend
npx http-server -p 3000 -o

## That's it! 
# Backend: http://localhost:5000
# Frontend: http://localhost:3000
